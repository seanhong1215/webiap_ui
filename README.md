# FlowRing — 企業智慧簽核平台

> 個人作品集展示版本 · Personal Portfolio Project

## 簡介

**FlowRing** 是一套以 **Vue 3** 建構的企業工作流程管理系統前端。本專案以一個完整的簽核申請系統為情境，展示以下技術能力：

- **角色權限系統**：使用者 / 管理員雙角色登入，路由守衛與功能隔離
- **大型資料呈現**：申請記錄分頁、多維度篩選、可展開詳情列
- **互動式管理後台**：流程分類分布圖、條件規則檢視、表單欄位管理
- **零 UI 依賴**：全部畫面以手寫 Vue 3 SFC + scoped SCSS 刻成，不依賴任何 UI 元件庫
- **Vuex 狀態管理**：登入狀態集中管理與 sessionStorage 持久化

> 本版本為**前端展示用途**，使用 Mock Data 模擬後端資料，無需任何後端服務即可完整體驗。

## 功能概覽

### 使用者功能

| 功能 | 說明 |
|------|------|
| **控制台** | 統計卡片（待審核、申請中、本月核准）、近期活動、快速申請入口 |
| **申請表單** | 分類瀏覽 8 種流程表單，點擊填寫並送出申請 |
| **審核中心** | 待我審核列表（含緊急標籤）、快速核准/退回、已處理歷史 |
| **我的申請** | 個人申請列表、狀態篩選、點擊展開完整審核軌跡 |

### 管理員後台

| 功能 | 說明 |
|------|------|
| **系統總覽** | 4 大統計卡片、流程分類分布圖（2×2 卡片格）、狀態說明欄、近期動態 |
| **流程總覽** | 8 個流程卡片、狀態 Tab 篩選、查看規則（含條件 CRUD）、管理欄位 |
| **申請記錄** | 跨使用者申請記錄查詢、多維度篩選、展開查看表單內容與審核軌跡 |

```bash

# 啟動開發伺服器

# 開啟瀏覽器
http://localhost:8080
```

## 技術棧

| 分類 | 技術 |
|------|------|
| 框架 | Vue 3 + Vue Router 4 + Vuex 4 |
| UI | 手寫 Vue 3 SFC + scoped SCSS（無 UI 元件庫依賴） |
| 圖示 | Font Awesome |
| 建構 | Vue CLI 4 + Webpack |
| 瀏覽器自動化 | Playwright（截圖測試） |
| 狀態持久化 | vuex-persistedstate（sessionStorage） |

> 執行期相依套件僅 6 個(`vue` / `vue-router` / `vuex` / `vuex-persistedstate` / `core-js` /
> `register-service-worker`),vendor bundle 約 46 KB(gzip)。

## 專案結構

```
src/
├── components/
│   └── Layout/         # AppShell（側邊欄 + 頂部列）
├── store/
│   ├── index.js        # Vuex + sessionStorage 持久化設定
│   └── modules/
│       └── user.js     # 登入使用者狀態（角色、姓名、部門）
├── utils/
│   ├── mockData.js     # Mock 資料（8 流程、12 筆申請記錄、2 帳號）
│   └── global.js       # processTypeConverter（依起訖日推導流程狀態）
└── views/
    ├── Login.vue           # 登入（帳密驗證）
    ├── Dashboard.vue       # 使用者控制台
    ├── FormCatalog.vue     # 申請表單目錄
    ├── FormDetail.vue      # 表單填寫頁
    ├── ApprovalCenter.vue  # 審核中心
    ├── MyRequests.vue      # 我的申請
    ├── AdminOverview.vue   # 管理後台 — 系統總覽
    ├── AdminProcesses.vue  # 管理後台 — 流程總覽
    └── AdminRecords.vue    # 管理後台 — 申請記錄
```

## Mock 資料設計

**8 個示範流程**，涵蓋 4 大分類與 4 種狀態：

| 分類 | 流程 | 狀態 |
|------|------|------|
| 人事管理 | 請假申請流程 | 已生效 |
| 人事管理 | 加班申請流程 | 待生效 |
| 財務管理 | 費用報銷申請 | 已生效 |
| 財務管理 | 採購申請流程 | 待生效 |
| 行政管理 | 會議室申請流程 | 已生效 |
| 行政管理 | 車輛申請流程 | 已關閉 |
| 資產管理 | 資源借用申請 | 已生效 |
| 資產管理 | 設備維修申請 | 已關閉 |

**流程狀態說明：**

| 狀態 | 說明 |
|------|------|
| 初始 | 已建立，尚未設定生效時間 |
| 待生效 | 已排程，等待生效日期到來 |
| 已生效 | 開放申請，使用者可提交 |
| 已關閉 | 已超過結束時間，不再受理 |
