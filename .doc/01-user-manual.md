## 1. 系統介紹

**IAP Flow** 是一套企業內部的簽核管理平台，解決以下問題：

| 傳統方式的問題 | IAP Flow 的解法 |
|---|---|
| 紙本表單容易遺失 | 全數位化，永久保存 |
| 不知道審核到哪個環節 | 即時追蹤審核進度 |
| 找不到表單要給誰簽 | 系統自動轉送給正確的審核者 |
| 審核者不在位置無法處理 | 在任何裝置上皆可操作 |

### 支援的申請類型

| 類別 | 表單 |
|------|------|
| 人事管理 | 請假申請、加班申請 |
| 財務管理 | 費用報銷、採購申請 |
| 行政管理 | 會議室申請、車輛申請 |
| 資產管理 | 資源借用、設備維修 |

## 2. 登入系統

**網址**：開啟瀏覽器，前往公司提供的系統網址

**步驟**：

1. 在「帳號」欄位輸入您的員工帳號
2. 在「密碼」欄位輸入您的登入密碼
3. 點擊藍色「**登入系統**」按鈕
4. 系統自動帶您進入控制台

> 💡 **提示**：建議使用 Chrome 或 Edge 瀏覽器，螢幕寬度請保持 1024px 以上。

## 4. 提出申請

### 步驟一：找到表單

**方法 A**：從控制台點「**新增申請**」按鈕 → 進入表單目錄

**方法 B**：從左側導覽列點「**申請表單**」

您會看到所有可申請的表單，分為四個類別：
- 人事管理 / 財務管理 / 行政管理 / 資產管理

每張表單卡片會顯示：
- **表單名稱**與簡介
- **狀態標籤**：可申請（綠色）、即將生效（橘色）、草稿（灰色）、已停用（紅色）
- **版本**與**負責人員**

> ⚠️ **只有標示「可申請」的表單才能送出！**

### 步驟二：搜尋表單（可選）

- 在右上角搜尋框輸入關鍵字，例如「請假」
- 或點分類標籤篩選

### 步驟三：填寫表單

1. 點擊表單卡片的「**申請 →**」按鈕
2. 依序填寫每個欄位（標有 **\*** 的為必填）
3. 右側可以看到「**審核流程**」— 說明您的申請會經過哪些關卡

**各欄位類型說明**：
| 欄位外觀 | 如何操作 |
|---|---|
| 下拉選單 | 點擊後從選項中選擇 |
| 數字輸入 | 直接輸入數字，例如請假天數 |
| 日期選擇 | 點擊後會出現日曆，點選日期 |
| 文字區域 | 可以輸入多行文字說明 |

### 步驟四：送出申請

- 所有必填欄位填寫完畢後，右下角的「**送出申請**」按鈕會變為可點擊
- 點擊送出後，系統會顯示：
  - ✅ 成功圖示
  - 您的**申請編號**（格式如 `LEAVE-2026-0602-123`）
  - 接下來可選擇「查看我的申請」或「返回表單目錄」

> 📌 **申請編號很重要！** 請記下來，您可以用它和審核人員溝通。

## 5. 追蹤我的申請

從左側導覽列點「**我的申請**」

### 申請列表說明

每筆申請會顯示：
- **表單名稱**與申請編號
- **狀態徽章**（見下表）
- **目前關卡**（待審核時顯示）
- **步驟進度條**（橫向顯示目前在哪個關卡）

### 狀態說明

| 顏色 | 狀態 | 意思 |
|------|------|------|
| 🟡 橘色 | 待審核 | 正在等待審核人員處理 |
| 🟢 綠色 | 已核准 | 申請已通過，流程結束 |
| 🔴 紅色 | 已退回 | 審核人員退回，需要修改後重新申請 |

### 查看詳情

點擊任一申請卡片，右側會滑出「詳情面板」，顯示：
1. **審核進度時間軸** — 每個關卡的處理者、時間與備註
2. **申請內容** — 您當時填寫的所有資料

### 重新申請（申請被退回時）

若申請被退回，詳情面板底部會出現「**重新申請**」按鈕，點擊後自動帶入原始資料，方便您修改後重送。

### 篩選功能

列表頂部有篩選按鈕：
- 「全部」— 顯示所有申請
- 「待審核 X」— 只顯示處理中
- 「已核准」— 只顯示已完成
- 「已退回 X」— 只顯示需要處理的

## 6. 審核他人申請

**您會在以下情況收到通知**：
- 頂部鈴鐺圖示旁出現**紅色數字**
- 左側「審核中心」旁出現**紅色數字**

### 查看待審核

從左側導覽列點「**審核中心**」，預設顯示「**待我審核**」分頁。

每筆申請卡片顯示：
- 申請人姓名與部門
- 申請編號與日期
- **表單摘要**（前三個欄位值）
- 若有「**緊急**」紅色標籤，請優先處理

### 核准申請

**快速核准**：直接點卡片右側的「✓ **核准**」按鈕

**詳細審核**：點申請卡片本身，會彈出審核視窗，顯示：
- 完整申請內容
- 輸入審核備註（選填）
- 底部選擇「核准通過」或「退回申請」

### 退回申請

點「✕ **退回**」按鈕或審核視窗底部的退回按鈕。
退回後申請人會收到通知，可重新修改申請。

### 查看已處理紀錄

點「**已處理紀錄**」分頁，可以看到您過去的核准/退回歷史。

## 7. 常見問題

**Q：我的申請送出後要多久才會被審核？**
A：這取決於您的審核人員。系統只提供工具，不限制審核時間。

**Q：我可以撤回已送出的申請嗎？**
A：目前版本不支援撤回，請在送出前仔細確認填寫內容。

**Q：表單顯示「已停用」，我找不到我需要的表單怎麼辦？**
A：請聯繫系統管理員重新啟用該表單，或確認是否有替代表單可用。

**Q：我同時是申請人也是審核人怎麼辦？**
A：系統不會讓您審核自己的申請，該關卡會自動由上一層主管處理。

**Q：找不到想要申請的表單？**
A：使用搜尋功能（輸入關鍵字），或聯繫 HR / 行政部確認是否有新增表單需求。
