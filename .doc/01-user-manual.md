## 1. 系統介紹

**FlowRing** 是一套企業內部的智慧簽核管理平台，解決以下問題：

| 傳統方式的問題 | FlowRing 的解法 |
|---|---|
| 紙本表單容易遺失 | 全數位化，永久保存 |
| 不知道審核到哪個環節 | 即時追蹤審核進度 |
| 找不到表單要給誰簽 | 系統自動轉送給正確的審核者 |
| 審核者不在位置無法處理 | 在任何裝置上皆可操作 |

### 支援的申請類型

| 類別 | 表單 |
|------|------|
| 人事管理 | 請假申請、加班申請 |
| 財務管理 | 費用報銷、採購申請 |
| 行政管理 | 會議室申請、車輛申請 |
| 資產管理 | 資源借用、設備維修 |

## 2. 登入系統

**步驟：**

1. 開啟瀏覽器，前往系統網址
2. 在「帳號」欄位輸入您的員工帳號
3. 在「密碼」欄位輸入您的登入密碼
4. 點擊「**登入系統**」按鈕
5. 系統自動帶您進入控制台

> 登入頁下方顯示展示帳號提示，可快速確認帳密格式。

**展示帳號（作品集版本）：**

| 角色 | 帳號 | 密碼 |
|------|------|------|
| 一般使用者 | `user` | `123456` |
| 管理員 | `admin` | `admin` |

## 3. 控制台

登入後的第一個畫面，一眼掌握今天最重要的事。

### 上方歡迎區

- 顯示您的姓名與待審核提示
- 右側快捷按鈕：「**新增申請**」「**審核中心**」（紅色數字代表待處理件數）

### 四大統計卡片

| 卡片 | 說明 |
|------|------|
| 待我審核 | 等待您批核的件數 |
| 申請處理中 | 您送出但尚未完成的申請 |
| 本月已核准 | 本月系統共核准件數 |
| 平均審核時數 | 平均多久處理完一件申請 |

### 最近活動

顯示最新 5 筆動態（您的申請狀態變化、他人送來的待審核件）。

### 快速申請

點擊右下角「快速申請」區域的任一表單名稱，直接跳到填寫頁面。

## 4. 提出申請

### 步驟一：找到表單

**方法 A**：控制台點「**新增申請**」→ 進入表單目錄

**方法 B**：左側導覽列點「**申請表單**」

您會看到以分類排列的所有可申請表單（人事 / 財務 / 行政 / 資產）。

### 步驟二：點擊表單

點擊表單卡片上的「**申請 →**」按鈕進入填寫頁。

### 步驟三：填寫送出

1. 依序填寫每個欄位
2. 右側可查看此表單的「審核流程」說明
3. 填寫完畢後點「**送出申請**」

送出後系統顯示申請編號（如 `LEAVE-2026-0602-123`），請記下備查。

## 5. 追蹤我的申請

從左側導覽列點「**我的申請**」。

### 篩選功能

列表頂部可按狀態篩選：全部 / 待審核 / 已核准 / 已退回

### 申請狀態說明

| 狀態 | 意思 |
|------|------|
| 🟡 待審核 | 正在等待審核人員處理 |
| 🟢 已核准 | 申請已通過，流程結束 |
| 🔴 已退回 | 審核人員退回，請修改後重新申請 |

### 查看詳情

點擊任一申請列，下方展開詳情：
- **表單內容**：您當時填寫的所有資料
- **審核軌跡**：每個關卡的處理者、時間與備註

## 6. 審核他人申請

當有待審核件時，左側「審核中心」旁會出現紅色數字提示。

### 待我審核

點「**審核中心**」，預設顯示「待我審核」列表。

每筆申請卡片顯示：
- 申請人姓名與部門
- 申請編號與日期
- 表單摘要資料
- **緊急**紅色標籤（請優先處理）

**快速核准**：點卡片右側的「✓ 核准」按鈕

**退回申請**：點「✕ 退回」按鈕

**詳細審核**：點申請卡片本身，彈出完整審核視窗，可輸入審核備註再做決定。

### 已處理紀錄

點「**已處理紀錄**」分頁，查看您過去的核准 / 退回歷史。

## 7. 常見問題

**Q：我的申請送出後要多久才會被審核？**
A：這取決於審核人員的處理速度，系統不限制審核時間。

**Q：我可以撤回已送出的申請嗎？**
A：目前版本不支援撤回，送出前請仔細確認填寫內容。

**Q：找不到需要的表單怎麼辦？**
A：使用搜尋功能輸入關鍵字，或聯繫系統管理員確認表單是否需要開啟。

**Q：申請被退回後怎麼辦？**
A：在「我的申請」找到該申請，查看審核人員留的退回原因，修改後重新提交。
