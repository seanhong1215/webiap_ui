## 1. 管理員身份說明

管理員與一般使用者的差異：

| 功能 | 一般使用者 | 管理員 |
|------|---------|------|
| 控制台 | ✓ | ✓ |
| 申請表單 | ✓ | ✓ |
| 審核中心 | ✓ | ✓ |
| 我的申請 | ✓ | ✓ |
| **系統總覽** | ✗ | ✓ |
| **流程總覽** | ✗ | ✓ |
| **申請記錄** | ✗ | ✓ |

管理員登入後，左側導覽列會額外顯示「**管理後台**」區塊，頂部右側也會顯示「管理員」識別標籤。

> 展示帳號：帳號 `admin`、密碼 `admin`

## 2. 登入管理後台

1. 前往登入頁，輸入管理員帳號密碼
2. 登入後進入控制台（與一般使用者相同）
3. 點擊左側「**系統總覽**」進入管理後台

## 3. 系統總覽

管理後台的入口頁，提供全系統的快速概覽。

### 四大統計卡片

| 卡片 | 說明 |
|------|------|
| 已生效流程 | 目前開放申請的流程數量 |
| 本月申請總量 | 所有使用者本月送出的申請件數 |
| 本月核准件數 | 本月審核通過的件數（含核准率） |
| 目前待審核數 | 全系統尚未處理的待審核件數 |

### 流程分類分布

以 **2×2 卡片格**顯示各分類（人事 / 財務 / 行政 / 資產）的流程數量分布。

每張分類卡片顯示 4 個狀態的流程數量：

| 狀態 | 顏色 | 說明 |
|------|------|------|
| 初始 | 淡紫 | 已建立，尚未設定生效時間 |
| 待生效 | 紫色 | 已排程，等待生效日期到來 |
| 已生效 | 綠色 | 開放申請，使用者可提交此流程 |
| 已關閉 | 灰色 | 已超過結束時間，不再受理新申請 |

### 近期申請動態

顯示最新 6 筆跨使用者的申請動態（不限於登入者本人）。

## 4. 流程總覽

管理所有流程的中心，每個流程以卡片形式呈現。

### 頁面說明橫幅

頁面頂部有可收合的說明區塊，解釋「流程」「條件規則」「表單欄位」三者的關係：

- **流程**：一種簽核申請路徑（如請假申請流程）
- **條件規則**：依申請內容決定走哪條審核路徑
- **表單欄位**：申請人需要填寫的資料欄位

### 狀態 Tab 篩選

```
全部 | 已生效 | 待生效 | 初始 | 已關閉
```

點擊各 Tab 快速過濾，旁邊數字顯示該狀態的流程數量。

### 流程卡片資訊

每張卡片顯示：
- 流程名稱 + 狀態標籤
- 分類 / 版本 / 管理員
- 本月申請量 / 條件規則數 / 表單欄位數
- 生效時間範圍

### 管理條件規則

點擊「**管理條件規則**」按鈕，彈窗顯示該流程的所有條件規則，並提供完整 CRUD 操作：

**查看規則**：規則名稱 + 條件表達式（支援 `==`、`!=`、`>`、`<`、`&&`、`||`）

**新增規則**：
1. 點右上角「+ 新增規則」
2. 填寫規則名稱（必填）
3. 填寫條件表達式（選填，留空表示所有申請皆適用）
4. 點「新增」確認

**編輯規則**：點規則右側「編輯」圖示 → 原地展開輸入框 → 修改後點「儲存」

**刪除規則**：點規則右側「刪除」圖示 → 立即移除

### 管理表單欄位

點擊「**管理表單欄位**」按鈕，彈窗顯示該流程表單的所有欄位：

**新增欄位**：
1. 點右上角「+ 新增欄位」
2. 填寫欄位名稱（英文，程式碼用）
3. 填寫顯示說明（中文，顯示給申請人看）
4. 點「新增」確認

**編輯欄位**：點欄位右側「編輯」圖示 → 原地展開輸入框 → 修改後點儲存符號

**刪除欄位**：點欄位右側「刪除」圖示 → 立即移除

## 5. 申請記錄

查詢所有使用者的申請記錄（管理員獨有功能）。

### 摘要統計卡

頁面頂部顯示四張摘要卡，**可點擊快速篩選**：

| 卡片 | 功能 |
|------|------|
| 全部記錄 | 顯示所有申請 |
| 待審核 | 只顯示尚未完成審核的申請 |
| 已核准 | 只顯示審核通過的申請 |
| 已退回 | 只顯示被退回的申請 |

再次點擊已選中的卡片可取消篩選。

### 篩選列

| 篩選項 | 說明 |
|--------|------|
| 流程分類 | 人事 / 財務 / 行政 / 資產 |
| 申請部門 | 依申請人的所屬部門篩選 |
| 搜尋 | 輸入申請人姓名、申請序號或流程名稱 |

右側「重設篩選」按鈕清除所有條件（有篩選時才顯示）。

### 申請記錄表格

欄位說明：序號 / 申請人（含頭像）/ 部門 / 流程名稱 / 狀態 / 申請時間 / 目前步驟

**點擊任一記錄列展開詳情**：

左半部 — **表單內容**：顯示申請人當時填寫的所有欄位值

右半部 — **審核軌跡**：
- 時間線顯示每個審核關卡
- 🟢 綠色點：已完成
- 🟣 紫色點：進行中
- 🔴 紅色點：已退回
- 每個步驟顯示：關卡名稱、處理人員、處理時間、審核備註
